# Bank Fraud and Anomaly Detection

## Project Overview
This project focuses on detecting fraudulent banking transactions using Machine Learning techniques. The system analyzes transaction data and identifies suspicious or abnormal activities that may indicate fraud.

The goal of this project is to build a model that can automatically detect fraudulent transactions and help financial institutions reduce financial losses.

## Objectives
- Detect fraudulent transactions from banking data
- Identify abnormal patterns in transaction behavior
- Improve fraud detection accuracy using Machine Learning models
- Provide insights through data analysis

## Technologies Used
- Power BI (for visualization)
- Python
- Pandas
- XGBoost
- PySpark


## Dataset
The dataset contains banking transaction information such as:
- Transaction ID
- Transaction Amount
- Transaction Type
- Location
- Device Used
- Time of Transaction
- Fraud Label

The dataset is used to train machine learning models to classify transactions as **fraudulent or legitimate**.

## Methodology
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Fraud Detection
6. Evaluation of Model Performance

## Machine Learning Models Used
- Random Forest
- Isolation Forest
- XGBoost

These models help detect unusual transaction patterns and classify fraudulent activities.

## Project Workflow
1. Load transaction dataset
2. Clean and preprocess the data
3. Perform exploratory data analysis
4. Train machine learning models
5. Detect fraudulent transactions
6. Evaluate model accuracy

## Results
The model successfully identifies fraudulent transactions by analyzing transaction patterns and anomalies in the dataset.
